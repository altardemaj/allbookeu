How these rules work

These rules apply before a guest can confirm a booking.

Tables still matter If your tables are set up, AllBookEU still checks table capacity and existing assignments.
Slot limits are global Max reservations and max guests count all active bookings at the same time.
Buffer is table-based A 60 minute buffer keeps the same table from being booked within 60 minutes of another active booking.
Lead time is guest-facing Same-day times inside the lead-time window disappear from the public booking widget.
